    Remuxing DVDs¶

    Guide originally by kurzisztk

    Tools You'll Need¶

    • MakeMKV (latest release)
    • eac3to + updated libraries
    • MKVToolNix
    • VLC (or any player capable of reading full DVD menus)

    Getting Started¶

    MakeMKV or eac3to — Which One Should You Use?¶

    Primary tool: MakeMKV

    • Fast
    • Easy
    • Preserves metadata
    • Usually no need to use MKVToolNix unless adding subtitles or tagging

    eac3to

    • Better for Blu-ray
    • Has advanced demuxing features
    • Less convenient for DVDs
    • Often unnecessary unless extracting PCM or dealing with complex discs

    Identifying Whether a DVD Is Retail or Custom¶

    Not all DVD rips are usable. Avoid anything that is:

    • DVDR (except Amazon MOD discs)
    • DVDRip
    • DVDShrink’d ISOs
    • DVDFab v1.x rips
    • MacTheRipper rips

    These often cause:

    • Corrupted VOBs
    • Missing frames
    • Artifacting
    • Broken navigation

    MakeMKV attempts detection but cannot always verify authenticity.

    Important Advice for Rutracker DVDs¶

    Look for these indicators:

    Russian Phrase Meaning Action
    Перевод 1: Профессиональный Professional dub Likely OK
    Оригинальная аудиодорожка: нет Original audio removed ❌ Avoid
    Качество: DVD5 (сжатый) Transcoded DVD9→DVD5 ❌ Avoid
    (custom) Modified disc ❌ Avoid

    Also check audio track descriptions. Fan dubs (common for Russian releases) should never be included in a remux.

    Audio Track Labeling¶

    DVD Audio Label in MKV Notes
    Dolby Digital 1.0/2.0/5.1 AC‑3 1.0 / 2.0 / 5.1 Standard DVD audio
    PCM 1.0 / 2.0 FLAC 1.0 / 2.0 Convert PCM → FLAC first
    DTS 5.1 DTS 5.1 Only include if better than AC‑3

    Converting PCM → FLAC¶

    Option 1 — Using eac3to directly on VOBs:

    eac3to VOB_01_1.VOB+VOB_01_2.VOB ... 3: output.pcm
eac3to output.pcm output.flac

    Option 2 — Extract from MakeMKV-generated MKV:

    eac3to movie.mkv -demux

    Choosing the Best Audio Track¶

    Some DVDs contain both DTS and AC‑3 versions. Don't include both.

    Evaluate by:

    • Spectral Analysis:
      Understanding Spectral Analysis
    • Listening tests

    Bitrate alone does not determine quality.

    Subtitle Track Labeling¶

    Type Meaning
    English Standard English subtitles
    English (SDH/HOH) Includes sound cues
    English (CC) Text-based closed captions (MakeMKV marks them as text)
    Other languages Use English names: “French”, “Spanish”, etc.

    If multiple dialects exist:

    • Spanish (Latin American)
    • Spanish (Castilian)
    • Portuguese vs Brazilian Portuguese

    Make sure to label them correctly.

    Forced Default Rules¶

    • Foreign-language movie → English should be track #1, set to Default + Forced.
    • English movie → Only set Forced if the disc includes a true foreign-dialogue forced track.

    Other Track Types¶

    Commentary Tracks¶

    Label like:

    • Commentary by Director Jane Doe
    • Commentary by Cast Members A, B, C

    If foreign films include commentaries in other languages, keep them.

    Music-Only / Orchestral¶

    Set language to zxx:

    • AC-3 2.0 (Orchestral Score by Hans Zimmer)
    • FLAC 1.0 (Music Only)

    Foreign Dubs¶

    ❌ Do NOT include fan-made dubs
    ❌ Do NOT include non-original-language dubs unless English

    Chapters¶

    Use DVD menus to determine:

    • Chapter names
    • Chapter counts
    • Correct sequence

    In MakeMKV, use the Name field (not metadata language).

    File Naming Standards¶

    Movies¶

    Folder:

    Example.Movie.1990.PAL.DVD.DD.2.0.MPEG-2.REMUX-YourName

    File:

    Example.Movie.1990.PAL.DVD.DD.2.0.MPEG-2.REMUX-YourName.mkv

    TV Shows¶

    Folder:

    Seinfeld.S01.NTSC.DVD.DD.2.0.MPEG-2.REMUX-YourName

    Episode files:

    Seinfeld.S01E01.Good.News.Bad.News.NTSC.DVD.DD.2.0.MPEG-2.REMUX-YourName.mkv

    Avoid special characters.

    Tagging IMDB, TMDB, and TVDB Metadata¶

    Follow this guide:

    ➡️ Advanced Metadata

    Each MKV should include:

    • IMDB ID
    • TMDB ID
    • TVDB ID (for TV shows)

    Subtitle OCR Recommendations¶

    Best tools:

    • SubRip
    • SubExtractor (recommended)

    Not recommended:

    • Subtitle Edit (for OCR only — good for editing afterward)

    After OCR:

    • Load into Subtitle Edit
    • Fix spacing, punctuation, and OCR errors
    • Save to SRT
    • Merge into MKV using MKVToolNix
    • Label as: English (CC) if applicable
