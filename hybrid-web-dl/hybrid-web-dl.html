    Hybrid WEB-DL Dolby Vision (DV + HDR10/HDR10+ Fallback) Guide¶

    Goal: create Hybrid WEB-DL releases by combining Dolby Vision (DV) with HDR10 or HDR10+ streams to produce Profile 8.1 DV with HDR fallback, matching how platforms like HMAX deliver DV.HDR natively.

    What You'll Learn

    • What Hybrid WEB-DL is and when you need it
    • How to create DV.HDR hybrid video using DDVT
    • Tools required (DDVT, dovi_tool, MKVToolNix)
    • Best practices + naming convention link

    1. What Is a Hybrid WEB-DL?¶

    A Hybrid WEB-DL merges:

    • A Dolby Vision (DV) enhancement layer
    • A HDR10 or HDR10+ base layer

    This produces a single-layer DV Profile 8.1 stream with HDR fallback, commonly labeled:

    DV.HDR or DV.HDR10Plus

    Hybrid WEB-DLs are needed when:

    • A service provides DV-only (no HDR fallback)
    • DV and HDR versions differ regionally
    • DV lacks a usable HDR base layer
    • You want support for both DV and non-DV displays

    2. When Hybrid WEB-DL Is Needed¶

    Provider Hybrid Needed? Notes
    HMAX ❌ No DV and DV.HDR already provided.
    Hulu ❌ No DV and DV.HDR10+ already provided.
    Netflix ✔ Yes DV and HDR provided separately.
    Disney+ ✔ Yes DV and HDR provided separately.
    Amazon ✔ Yes DV and HDR10+ provided separately.
    Apple TV+ ✔ Yes DV and HDR10+ provided separately.

    Notes¶

    • DSNP US uses the same Hulu hybrid DV/HDR10+ structure for Hulu‑licensed titles.

    4. DDVT Overview¶

    DDVT bundles Dolby Vision utilities and performs:

    • DV RPU extraction
    • HDR10+ → DV metadata conversion (experimental)
    • Profile 5/7 → Profile 8.1 conversion
    • RPU injection into HDR10/HDR10+
    • Delay correction
    • Batch automation

    Credits:
    quietvoid (dovi_tool)
    Atak_Snajpera (BorderDetect)

    5. Workflow Summary (Recommended Method)¶

    Use DDVT – Fully Automated Hybrid Creation¶

    This is the recommended workflow for Hybrid WEB-DL creation.

    Steps:¶

    1. Download DDVT:¶

    https://github.com/DonaldFaQ/DDVT

    2. Place your DV stream and HDR10/HDR10+ stream into the INPUT folder.¶

    3. Run:¶

    RUN_ME.bat

    4. DDVT automatically:¶

    • Extracts DV RPU
    • Converts DV to Profile 8.1
    • Extracts HDR10/HDR10+
    • Injects DV metadata into HDR
    • Fixes delays
    • Outputs:
    dv_hybrid.hevc

    6. Mux Final Hybrid WEB-DL¶

    Use MKVToolNix:

    • Video → dv_hybrid.hevc
    • Audio → Original WEB-DL audio
    • Subtitles → Clean and tagged
    • Chapters → Clean or remove

    For naming conventions, follow:

    👉 Best Practices – Naming Conventions

    This ensures Hybrid releases follow consistent patterns like DV.HDR, DV.HDR10Plus, provider tags, etc.

    7. Tips & Best Practices¶

    Verify DV RPU metadata

    dovi_tool info -s RPU_p81.bin

    Do NOT mix mismatched DV & HDR sources

    Both streams must align perfectly frame‑for‑frame.
