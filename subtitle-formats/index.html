    Subtitle Formats Explained¶

    WEBVTT / VTT (Web Video Text Tracks)¶

    Used By:
    Netflix, Disney+, Hulu, HBO Max, Prime Video, Apple TV+, YouTube

    Characteristics:
    - Plain-text subtitle format
    - Very simple styling
    - No typesetting, no vectors
    - Supports basic placement & italics
    - Millisecond accuracy

    Pros:
    - Lightweight
    - Easy to convert
    - Great for SRT conversion
    - Supported everywhere

    Cons:
    - Cannot preserve styling
    - No outlines, borders, colors
    - Bad for anime signs

    Ideal Usage:
    - WEB-DL cleanup
    - Converting to SRT or ASS

    SRT (SubRip)¶

    Used By:
    Amazon WEB-DLs, older rips, international releases

    Characteristics:
    - Plain-text format
    - No styling beyond italics
    - Universally supported

    Pros:
    - Easy to edit
    - Universal compatibility
    - Perfect for clean captions

    Cons:
    - No advanced formatting
    - Cannot style signs or dialogue

    Ideal Usage:
    - Dialogue-only subtitles
    - Converting VTT → SRT

    ASS (Advanced SubStation Alpha)¶

    Used By:
    Anime fansubs, Crunchyroll, HIDIVE

    Characteristics:
    - Full styling engine
    - Fonts, outlines, blur, shadows
    - Precise placement
    - Vector drawing & motion

    Pros:
    - Best format for typesetting
    - Necessary for anime signs
    - Supports attached fonts

    Cons:
    - Requires Aegisub to edit
    - Must embed fonts for playback

    Ideal Usage:
    - Crunchyroll subtitle restoration
    - Typesetting & sign styling
    - Anime releases

    SSA (SubStation Alpha)¶

    Older, simplified version of ASS.

    Pros:
    - Works for legacy fansubs

    Cons:
    - Missing many ASS features
    - No vector support

    Ideal Usage:
    - Only when restoring retro subs

    PGS / SUP (Blu-ray Subtitles)¶

    Used By:
    Blu-ray, UHD Blu-ray

    Characteristics:
    - Bitmap images
    - Not text-based
    - Requires OCR for editing

    Pros:
    - Perfect reproduction
    - Includes forced subs

    Cons:
    - Hard to edit
    - Large file sizes

    Ideal Usage:
    - Blu-ray remuxing
    - OCR → SRT/ASS workflows

    VOBSUB (SUB/IDX)¶

    Used By:
    DVDs, early streaming encodes, older anime releases

    Characteristics:
    - Bitmap images (subpictures)
    - Includes .sub (image) + .idx (palette + timing)
    - Lower resolution (DVD era)
    - Limited color palette

    Pros:
    - Exact reproduction of DVD subtitles
    - Easier OCR than PGS
    - MKV-compatible

    Cons:
    - Must be OCR’d to text
    - Low resolution on HD/4K
    - Palette issues sometimes occur

    Ideal Usage:
    - OCR → SRT for dialogue
    - OCR → ASS for styled signs
    - DVD restoration workflows

    SCC (Scenarist Closed Captions)¶

    Used By:
    DVD, broadcast, older streaming content

    Characteristics:
    - Closed-caption formatting
    - 29.97 fps timecodes
    - Limited styling

    Pros:
    - Retains CC structure
    - Converts to SRT

    Cons:
    - Often messy formatting
    - Requires cleanup

    Ideal Usage:
    - Converting broadcast captions → SRT

    TTML / XML / DFXP (Timed Text Markup Language)¶

    Used By:
    Amazon Prime Video, HBO Max, Apple TV+

    Characteristics:
    - XML-based
    - Supports regions + basic styling
    - Converts easily via Subtitle Edit

    Pros:
    - Better styling than SRT
    - Good forced-track support
    - Easy conversion

    Cons:
    - Styling ignored by many players
    - Too limited for anime signs

    Ideal Usage:
    - TTML → SRT
    - Extracting forced signs

    EIA-608 / EIA-708 (Closed Captioning)¶

    Used By:
    TV broadcasts, cable channels

    Characteristics:
    - Encoded CC data
    - Limited styling
    - Often noisy/messy

    Pros:
    - Sometimes the only subtitle source
    - Converts to SRT

    Cons:
    - Requires major cleanup

    Ideal Usage:
    - CC → SRT for remuxing
    - Fixing legacy broadcast content

    Which Subtitle Format Should You Use?¶

    Use Case Best Format
    Clean dialogue subs SRT
    Anime, signs, typesetting ASS
    Streaming subs VTT → SRT
    Blu-ray subs PGS → OCR → SRT/ASS
    DVD subs VOBSUB → OCR → SRT/ASS
    Broadcast captions SCC → SRT
    Crunchyroll restoration ASS + fonts
    Forced-only subs SRT or VTT

    Tools for Working With Subtitle Formats¶

    Subtitle Edit
    - VTT → SRT
    - TTML → SRT
    - PGS → OCR → SRT/ASS
    - SUB/IDX → OCR → SRT/ASS
    - SCC → SRT
    - Cleanup: SDH, italics, timing

    Aegisub
    - ASS typesetting
    - Font recreation
    - Sign styling
    - Timing correction
