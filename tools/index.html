    Tools & Software¶

    A complete collection of all recommended tools used throughout the Kitsune Starter Guide.
    Organized by category for easy navigation.

    What You'll Find

    • Video & Dolby Vision tools
    • Audio syncing and waveform tools
    • Subtitle editing and typesetting tools
    • File renaming and media organization
    • Automation and correction utilities

    1. Video / MKV / Dolby Vision Tools¶

    MKVToolNix¶

    The industry-standard tool for MKV muxing, remuxing, metadata editing, and embedding subtitles/fonts.

    Includes: - MKVToolNix GUI
    - mkvmerge (CLI)
    - mkvextract (extract video/audio/subtitles/fonts)
    - mkvpropedit (edit flags/metadata without remuxing)

    Use Cases: - Add/replace audio tracks
    - Apply audio delay offsets
    - Attach fonts for ASS subtitles
    - Reorder tracks and set default/forced flags
    - Clean metadata and remove attachments

    ➡️ https://mkvtoolnix.download/

    DDVT (Dolby Vision Toolset)¶

    Automates creation of DV Profile 8.1 Hybrid WEB-DL streams from DV + HDR sources.

    Use Cases: - Convert DV-only streams into DV.HDR
    - Inject RPU metadata
    - Convert Profile 5/7 → 8.1
    - Batch hybrid creation

    ➡️ https://github.com/DonaldFaQ/DDVT

    dovi_tool¶

    Powerful CLI tool for low-level Dolby Vision operations.

    Use Cases: - Extract/convert RPU
    - Inject RPU into HDR10/HDR10+ streams
    - Convert DV profiles
    - Analyze metadata

    ➡️ https://github.com/quietvoid/dovi_tool

    FFmpeg¶

    Versatile command-line framework used for video/audio processing.

    Use Cases: - Apply audio offsets (-itsoffset)
    - Extract audio/video streams
    - Convert HDR layers
    - Combine video + audio

    ➡️ https://ffmpeg.org/

    DaVinci Resolve / Adobe Premiere Pro¶

    Professional editors for visual waveform syncing and timeline comparison.

    Use Cases: - Fixing complex sync issues
    - Drift correction
    - Visual waveform matching

    ➡️ DaVinci Resolve
    ➡️ Adobe Premiere Pro

    MediaInfo¶

    Essential tool for inspecting container metadata, codec details, HDR/Dolby Vision flags, bitrates, chapters, audio formats, and track structure.

    Use Cases: - Verify HDR, HDR10+, and Dolby Vision profiles
    - Check audio codecs, bitrates, language tags, and channel layouts
    - Confirm subtitle formats
    - Identify mismatches between providers
    - Validate final mux integrity

    ➡️ https://mediaarea.net/en/MediaInfo

    StreamFab (Beginner-Friendly WEB‑DL Acquisition Tool)¶

    A popular tool for new users getting started with WEB‑DL workflows.
    StreamFab allows beginners to download content from supported streaming services using a simple, GUI‑based approach.

    Use Cases: - Easy WEB‑DL acquisition for beginners
    - GUI workflow with no command-line knowledge required
    - Works well with the workflows described in this guide
    - Frequently paired with MKVToolNix, Aegisub, and FFmpeg for post‑processing

    ➡️ Official Site: https://streamfab.com/
    ➡️ YouTube Tutorials: https://www.youtube.com/@StreamFabOfficial

    For Beginners

    StreamFab is widely used by people just starting out with WEB‑DL creation
    before transitioning into advanced workflows like DDVT, FFmpeg, and manual muxing.

    2. Audio Tools (Syncing, Offsets, Analysis)¶

    OffsetFinder¶

    Automatically determines audio delay between two tracks.

    Use Cases: - Auto-detect Crunchyroll vs Amazon audio offsets
    - Verify manual waveform comparisons
    - Ideal for WEB-DL audio sync workflows

    ➡️ OffsetFinder

    eac3to¶

    Advanced audio processing tool for delay shifting, demuxing, and remuxing.

    Use Cases: - Apply +ms or –ms delays
    - Fix AC3/EAC3/DTS tracks
    - Lossless delay correction

    ➡️ https://madshi.net/eac3to/

    Audacity¶

    Free waveform editor — perfect for spotting offsets visually.

    Use Cases: - Compare AAC vs DDP waveforms
    - Identify sync points
    - Confirm audio drift

    ➡️ https://www.audacityteam.org/

    MPV / MPC-HC / VLC¶

    Playback tools used to validate sync during testing.

    ➡️ MPV
    ➡️ MPC-HC
    ➡️ VLC

    3. Subtitle Editing & Typesetting Tools¶

    Subtitle Edit¶

    Essential for fixing subtitle timing, correcting LayoutRes, converting formats, and cleaning ASS/SRT.

    Use Cases: - Fix incorrect PlayRes / LayoutRes values
    - Remove SDH
    - Format conversion (VTT → SRT / ASS)
    - Timing correction
    - OCR for PGS subtitles
    - Fonts Collector (detects missing ASS fonts)

    ➡️ https://www.nikse.dk/subtitleedit

    Aegisub¶

    The best tool for ASS subtitle editing, typesetting, and sign styling.

    Use Cases: - Typesetting Crunchyroll-style ASS
    - Waveform/spectrum audio sync
    - Sign recreation (blur, shadows, transforms)
    - Style editing (fonts, outlines, margins)
    - Karaoke effects

    Aegisub version guidance

    • Typesetters: Use the arch1t3cht fork for modern ASS typesetting, motion, outlines, and typeset sign recreation.
    • Non-typesetters (timing, basic edits, cleanup): Use Aegisub 3.4.0.

    See upstream bug report:
    arch1t3cht/Aegisub issue 177

    Crunchyroll Fonts Pack¶

    A collection of commonly used CR typesetting fonts.

    ➡️ Crunchyroll Fonts
