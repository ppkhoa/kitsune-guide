    Metadata Tagging¶

    Goal: embed IMDb, TMDb, TVDB (and optionally MAL/AniDB) identifiers directly into MKV files for reliable media manager matching.

    What You'll Learn

    • Why embed IDs: Prevent mismatches in Plex, Jellyfin, and other media centers
    • XML structure: Target levels for movies, shows, seasons, and episodes
    • Import methods: MKVToolNix GUI, mkvmerge, and mkvpropedit
    • Verification: What to check in MediaInfo

    1. Why Embed Metadata¶

    Filenames alone are unreliable for identification. Embedded database IDs provide:

    • Accurate matching in Plex, Jellyfin, Emby, Kodi
    • Compatibility with Trakt, scrapers, and plugins
    • Permanent identification that survives renaming

    Embedded IDs Prevent Mismatches

    Bad matches in Plex/Jellyfin are common with filename-only identification. Embedded IDs fix this permanently.

    Highly recommended for all MKV-based releases: WEB-DL, WEBRip, HDTV, BluRay encodes, REMUXes, etc.

    Multiple IDs Recommended

    Include multiple database IDs (IMDb + TMDb + TVDB) because APIs differ in accuracy, accessibility, and artwork quality. Some databases may also disappear over time.

    Technical Background

    Matroska uses EBML, a binary form of XML. This method complies with the MKV standard as defined in draft-ietf-cellar-tags-04 (section 6.16).

    2. Target Type Values¶

    <TargetTypeValue> determines what level the ID applies to:

    Level Value
    Movie 50
    Episode 50
    Season 60
    TV Show 70

    3. XML Templates¶

    Movies¶

    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE Tags SYSTEM "matroskatags.dtd">
<Tags>
    <Tag>
        <Targets>
            <TargetTypeValue>50</TargetTypeValue>
        </Targets>
        <Simple>
            <Name>IMDB</Name>
            <String>tt1234567</String>
        </Simple>
        <Simple>
            <Name>TMDB</Name>
            <String>movie/123456</String>
        </Simple>
    </Tag>
</Tags>

    TV Shows (Basic)¶

    Series-level tagging only:

    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE Tags SYSTEM "matroskatags.dtd">
<Tags>
    <Tag>
        <Targets>
            <TargetTypeValue>70</TargetTypeValue>
        </Targets>
        <Simple>
            <Name>IMDB</Name>
            <String>tt1234567</String>
        </Simple>
        <Simple>
            <Name>TMDB</Name>
            <String>tv/123456</String>
        </Simple>
        <Simple>
            <Name>TVDB</Name>
            <String>123456</String>
        </Simple>
    </Tag>
</Tags>

    TV Shows (Advanced)¶

    Show + Season + Episode tagging:

    <?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE Tags SYSTEM "matroskatags.dtd">
<Tags>
    <Tag> <!-- TV Show -->
        <Targets>
            <TargetTypeValue>70</TargetTypeValue>
        </Targets>
        <Simple>
            <Name>IMDB</Name>
            <String>tt1234567</String>
        </Simple>
        <Simple>
            <Name>TMDB</Name>
            <String>tv/123456</String>
        </Simple>
        <Simple>
            <Name>TVDB</Name>
            <String>123456</String>
        </Simple>
    </Tag>
    <Tag> <!-- Season -->
        <Targets>
            <TargetTypeValue>60</TargetTypeValue>
        </Targets>
        <Simple>
            <Name>PART_NUMBER</Name>
            <String>1</String>
        </Simple>
        <Simple>
            <Name>TOTAL_PARTS</Name>
            <String>10</String>
        </Simple>
    </Tag>
    <Tag> <!-- Episode -->
        <Targets>
            <TargetTypeValue>50</TargetTypeValue>
        </Targets>
        <Simple>
            <Name>PART_NUMBER</Name>
            <String>10</String>
        </Simple>
        <Simple>
            <Name>IMDB</Name>
            <String>tt1234568</String>
        </Simple>
    </Tag>
</Tags>

    Anime Tags (Optional)¶

    Add MAL and AniDB identifiers:

    <Simple>
    <Name>MyAnimeList</Name>
    <String>368</String>
</Simple>
<Simple>
    <Name>AniDB</Name>
    <String>528</String>
</Simple>

    4. Import Methods¶

    Use before muxing your final file.

    In Output → Global tags, select your XML file.

    Mux a new file with metadata:

    mkvmerge input.mkv --global-tags file.xml -o output.mkv

    Edit an existing file without remuxing:

    mkvpropedit file.mkv -t global:file.xml

    5. Verification¶

    MediaInfo Output¶

    After importing, MediaInfo should show:

    IMDB: tt1234567
TMDB: tv/12345
TVDB: 123456

    For advanced tagging:

    Track name/Position: 1
Track name/Total: 8

    End Result
