    Khak1s' Guide to Demuxing a Blu-ray Disc¶

    Goal: extract individual streams (video, audio, subtitles, chapters) from Blu-ray disc structure using eac3to with proper error detection.

    What You'll Learn

    • eac3to setup: Installation and PATH configuration
    • Playlist identification: Finding the correct main feature playlist
    • Demuxing workflow: Extract all or specific tracks from Blu-ray
    • Special cases: Forced subtitles, distributor logos, seamless branching
    • Tool selection: When to use eac3to vs DGDemux vs MakeMKV

    Tool Selection¶

    Disc Type Recommended Tool Reason
    1080p Blu-ray eac3to Exceptional error detection and correction
    UHD Blu-ray (HEVC) DGDemux or MakeMKV Proper PTS timing for H.265
    Seamless branching DGDemux or MakeMKV Correct segment handling

    eac3to HEVC Limitation

    eac3to cannot process PTS timing on HEVC (H.265) video correctly. For Ultra HD Blu-rays (UHDBDs) that use seamless branching, use DGDemux or MakeMKV instead.

    Setting Up eac3to¶

    Step 1: Install eac3to¶

    Use the version bundled with StaxRip (includes updated codecs):

    ➡️ https://github.com/staxrip/staxrip/releases

    Step 2: Add to System PATH¶

    Add the eac3to folder to your system PATH to run from any Command Prompt.

    PATH Configuration

    Search online for "how to add folder to system PATH Windows" for detailed instructions specific to your Windows version.

    Demuxing Workflow¶

    Step 1: Open Command Prompt¶

    Navigate to your project directory:

    cd /d <projectFolderPath>

    Example: 

    cd /d "D:\workshop\"

    Step 2: Identify the Playlist¶

    Scan the Blu-ray to list available playlists:

    eac3to <BDMVFolderPath>

    Example: 

    eac3to "D:\discs\MovieTitle\BDMV"

    Step 3: Demux All Tracks¶

    Extract all tracks from a playlist:

    eac3to <BDMVFolderPath> <playlistNumber>) -demux -log="eac3to.txt"

    Example: 

    eac3to "D:\discs\MovieTitle\BDMV" 1) -demux -log="eac3to.txt"

    Step 4: Demux Specific Tracks¶

    Extract only selected tracks:

    eac3to <BDMVFolderPath> <playlistNumber>) <trackNumber>:<outputFilename> -log="eac3to.txt"

    Example: 

    eac3to "D:\discs\MovieTitle\BDMV" 1) 1:chapters.txt 2:video.* 3:audio.* -log="eac3to.txt"

    DTS Express Tracks

    Use .dts instead of .* for DTS Express tracks to ensure correct extraction.

    Identifying the Correct Playlist¶

    Blu-rays store playlists as .MPLS files pointing to .M2TS video segments. Use BDInfo to identify the correct main playlist:

    ➡️ https://www.videohelp.com/software/BDInfo

    Playlist Selection Criteria¶

    Criterion What to Check
    Main feature Usually the longest runtime
    Seamless branching Playlist references all M2TS segments
    Playback verification Open in VLC or MPC-HC to confirm
    Multiple cuts Theatrical vs extended versions

    Verification Step

    Always open the playlist in VLC or MPC-HC to verify correct ordering before demuxing. This catches issues with wrong playlists or missing segments.

    Special Considerations¶

    Logs and Warnings¶

    Situation Action
    Always Read eac3to.txt after demuxing
    Minor audio delays A few milliseconds is normal
    Errors or warnings Investigate before proceeding

    Forced Subtitles¶

    If forced subtitles are detected, extract with BDSup2Sub:

    Step Action
    1 Load the subtitle file
    2 Set correct FPS
    3 Export forced-only subtitle track

    Removing Distributor Logos¶

    If a playlist includes unwanted bumpers or logos, demux only the specific M2TS file:

    eac3to <M2TSFilePath> -demux -log="eac3to.txt"

    Example: 

    eac3to "D:\discs\MovieTitle\BDMV\STREAM\00001.m2ts" -demux -log="eac3to.txt"

    Alternative Tools¶

    Tool Best For Link
    eac3to 1080p Blu-ray demuxing Bundled with StaxRip
    DGDemux UHD Blu-ray, Dolby Vision ➡️ https://www.videohelp.com/software/DGDemux
    MakeMKV Quick remuxing, UHD support ➡️ https://www.makemkv.com
    BDInfo Playlist identification ➡️ https://www.videohelp.com/software/BDInfo
