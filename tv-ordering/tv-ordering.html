    TV Show Ordering Guide¶

    Goal: ensure episode numbering matches TVDB standards so releases work correctly in Sonarr, Plex, Jellyfin, and other media managers.

    What You'll Configure

    • Database standard: TVDB as primary, TMDB as fallback
    • Verification: Cross-check streaming platform order against TVDB
    • Renaming: Use FileBot for consistent, automation-friendly filenames
    • Special handling: Merged episodes, specials, and regional differences

    1. Why Streaming Platforms Are Wrong¶

    Most streaming services do NOT follow TVDB or TMDB ordering.

    Platform Common Issues
    Netflix Rearranges seasons between regions
    Disney+ Merges specials into standard episodes
    Amazon Randomly renumbers episodes between seasons
    Hulu Sometimes reorders entire arcs
    Crunchyroll Often ignores canonical anime numbering

    Never Trust Streaming Platform Order

    Incorrect ordering causes:

    • Wrong metadata matching
    • Duplicates or missing episodes
    • Incorrect subtitles
    • Tracker violations or nukes

    2. Database Standards¶

    TVDB vs TMDB¶

    Aspect TVDB TMDB
    Primary Use TV shows, anime Movies, TV shows
    Automation Support Sonarr, most tools Emerging support
    Tracker Expectation Standard Fallback only
    When to Use Always first When TVDB is wrong/incomplete

    TVDB First

    TVDB is the primary standard. Use TMDB only when TVDB is incomplete or incorrect.

    3. Verify and Correct Ordering¶

    Check TVDB¶

    Compare your WEB-DL files with TheTVDB:

    • Episode numbers
    • Episode titles
    • Air dates
    • Specials
    • Multi-part episodes

    Cross-check TMDB (Optional)¶

    Use TMDB only when:

    • TVDB is missing episodes
    • TVDB has incorrect numbering
    • Anime with complicated arcs

    Rename with FileBot¶

    FileBot automatically renames episodes based on TVDB ordering:

    filebot -rename *.mkv --db TheTVDB

    4. Common Problems¶

    Problem Description Solution
    Merged episodes Platform combined two episodes Split into proper episode numbers
    Specials in Season 1 Specials incorrectly numbered Move to Season 0 (specials folder)
    Multi-part mislabeled "Part 2" labeled as Episode 2 Correct to proper episode number
    Regional differences UK vs US ordering differs Use TVDB's primary ordering
    Anime inconsistencies CR/Funi/TVDB disagree Verify broadcast order and TVDB

    Anime Verification

    For anime, always verify: official broadcast order, TVDB anime listings, arc grouping, and specials/OVAs. Anime groups also tend to use AniDB for ordering but they will a lot of times break support for programs like sonarr.

    5. Final Checks Before Release¶

    Pre-Release Checklist

    • Episode numbers match TVDB
    • Specials correctly placed in Season 0
    • Multi-part episodes labeled correctly
    • No streaming-site reorder errors
    • Filenames clean and consistent
    • Subtitles match corrected episode order
