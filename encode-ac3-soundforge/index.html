    AC3 Encoding with Sony Sound Forge¶

    This guide was split out of the parent Advanced x264 Encoding Guide to make it easier to reference and keep up to date.

    Introduction¶

    AC3 (Dolby Digital) is a widely supported multichannel audio codec. On PTP, it is typically used for the main audio track of encodes with multichannel audio.

    While eac3to can transcode to AC3 using the aften library, there are concerns that its implementation may not produce optimal audio quality.
    For best results, Sony Sound Forge is recommended, as it includes a reference AC3 encoder.

    Installation¶

    • eac3to
    • Sony Sound Forge (available on RED and other sources)
      ⚠️ Versions later than 11 are reportedly broken for AC3 encoding.

    Usage¶

    1. Create a .w64 file from the lossless source¶

    .w64 is required to bypass WAV size limitations.

    🔗 WAV format limitations

    eac3to.exe e:\example\BDMV\PLAYLIST\01000.mpls
M2TS, 2 video tracks, 3 audio tracks, 5 subtitle tracks, 1:42:55, 62.856p
1: Chapters, 20 chapters
2: h264/AVC, 1080p24 /1.001 (16:9)
3: h264/AVC, 480p24 /1.001 (20:11)
4: DTS Master Audio, English, 5.1 channels, 16 bits, 48kHz, -11ms
   (core: DTS, 5.1 channels, 1509kbps, 48kHz)

eac3to e:\example\BDMV\PLAYLIST\01000.mpls 4: e:\example\source\output.w64

    2. Analyze dialog loudness in Sound Forge¶

    1. Open output.w64 in Sony Sound Forge
    2. Wait for waveform processing to complete
    3. Select a region only in the Center channel containing primarily dialog

    ⚠️ Selecting all channels will average them and give an incorrect RMS value.

    1. Go to Process → Normalize

    1. Select Average RMS level (loudness) → Scan Levels
    2. Write down the RMS value
    3. Click Cancel (do NOT normalize)

    3. Encode AC3¶

    1. Go to File → Save As
    2. Select:
    3. Dolby Digital AC-3 Pro
    4. 5.1 Surround DVD profile

    1. Click Custom…

    Encoder settings¶

    • Dialog normalization:
    • Use measured RMS value, or -31 dB if lower
    • Bitrate:
    • 640 kbps → 1080p / 720p
    • 448 kbps → SD encodes

    4. Preprocessing settings¶

    Navigate to the Preprocessing tab:

    Enable: - LFE low-pass filter - Bandwidth low-pass filter

    Disable: - RF overmodulation protection - All other options

    Set Line Mode Profile to Film: Light

    About Line Mode (DRC)¶

    Line mode is Dynamic Range Control metadata.
    It does not modify the audio stream but embeds playback recommendations for receivers.

    • Headphones / 2.0 systems benefit if enabled
    • Full surround users can disable it in their AVR

    Note: Original lossless tracks (e.g. TrueHD) usually include DRC metadata, which is lost when transcoding via eac3to.

    Film: Light Profile Details¶

    Show parameters - Max Boost: 6 dB (below –53 dB) - Boost Range: –53 to –41 dB (2:1 ratio) - Null Band Width: 20 dB (–41 to –21 dB) - Early Cut Range: –26 to –11 dB (2:1 ratio) - Cut Range: –11 to +4 dB (20:1 ratio) 🔗 [A Guide to Dolby Metadata (PDF)](https://www.dolby.com/us/en/technologies/a-guide-to-dolby-metadata.pdf)
