    Crunchyroll Subtitles — Fixing & Restoring Typesetting¶

    Goal: restore both old styled Crunchyroll ASS subtitles and modern stripped subtitles, including correcting LayoutRes issues, re‑applying CR’s classic styling, attaching fonts, and repairing signs/typesetting.

    What You'll Learn

    • How Crunchyroll originally formatted subtitles (full ASS typesetting)
    • How new CR subtitles differ and why they break
    • How to repair old + new subtitles properly
    • How to fix LayoutRes / PlayRes mismatches
    • How to attach and match Crunchyroll fonts
    • Best tools for subtitle editing & restoration

    1. Fixing Old Crunchyroll ASS Subtitles (Legacy Typeset)¶

    Crunchyroll’s older subtitles used full ASS styling with precise outlines, shadows, and font metrics.
    To restore them correctly, two things must be fixed:

    • Correct LayoutRes / PlayRes values
    • Correct legacy CR styles (Trebuchet MS, outline thickness, etc.)

    Required CR LayoutRes Settings (Critical for Old Subs)¶

    Old CR subtitles break if LayoutRes does not match PlayRes.

    Incorrect LayoutRes causes: shrinking text, wrong positioning, broken signs

    Always ensure LayoutRes equals PlayRes.

    Correct rule:

    PlayResX = LayoutResX
PlayResY = LayoutResY

    How to Fix LayoutRes (Subtitle Edit)¶

    1. Open the .ass file
    2. Go to File → Advanced Sub Station Alpha properties
    3. Make LayoutRes match PlayRes
    4. Save the file

    Example (from CR subtitle metadata)¶

    PlayResX: 640      LayoutResX: 640
PlayResY: 360      LayoutResY: 360

    Crunchyroll’s Old Typesetting Specs (1920×1080 Reference)¶

    These were the standard metrics CR used for years:

    Trebuchet MS
60pt size
105% width
White text
Black 7px border
Drop shadow: 100% opacity/spread, size 7, distance 0

    Use these values in Aegisub when repairing CR subtitles or rebuilding dialogue styles.

    2. New Crunchyroll Subtitles (2025–Present)¶

    Crunchyroll no longer uses full ASS typesetting.
    They now rely on plain-text subtitles generated via:

    Closed Caption Converter

    Problems with new CR subtitles:

    • No embedded fonts
    • No outlines or shadows
    • No styled signs
    • Poor scaling
    • Missing positioning
    • Heavy reliance on player fallback fonts

    Because of this, all modern CR subtitles need manual restoration to match old CR quality.

    3. Restoring Crunchyroll Subtitle Styling¶

    To fix both old and new CR subtitles, follow this workflow:

    1. Apply old CR font styling¶

    Use the Trebuchet MS specs listed in Section 1.

    2. Fix LayoutRes / PlayRes¶

    Mandatory for both old and new CR subtitles.

    3. Rebuild sign typesetting (Aegisub)¶

    New CR subtitles remove:

    • Sign positioning
    • Motion transforms
    • Blur layers
    • Color/gradient styling

    All must be restored manually in Aegisub for proper accuracy.

    4. Crunchyroll Fonts — Download & Usage¶

    Crunchyroll commonly used a set of fonts for signs and dialogue.

    Crunchyroll Fonts

    You must embed these fonts in the MKV to ensure proper playback.

    Identify which fonts your file requires¶

    • Subtitle Edit → Tools → Fonts Collector
    • Aegisub → warns for missing fonts

    5. Attaching Fonts in MKVToolNix¶

    1. Open MKVToolNix GUI
    2. Go to the Attachments tab
    3. Add all .ttf / .otf fonts used
    4. Remux the final file

    Missing fonts = broken subtitles

    Fallback fonts will destroy: line spacing, width, outlines, styles, and overall CR appearance.

    6. Tool Selection¶

    Task Recommended Tool
    LayoutRes fixes Subtitle Edit
    Timing adjustments Subtitle Edit
    Format conversion Subtitle Edit
    Quick cleanup Subtitle Edit
    Sign typesetting Aegisub
    Style work Aegisub
    Font overrides Aegisub
    Karaoke & motion Aegisub
    Final muxing (fonts + subs) MKVToolNix

    Aegisub version guidance

    • Typesetters: Use the arch1t3cht fork for modern ASS typesetting, motion, outlines, and typeset sign recreation.
    • Non-typesetters: Use Aegisub 3.4.0for timing, basic edits, cleanup.

    Newer builds currently have a confirmed regression affecting non-typesetting workflows.
    See: arch1t3cht/Aegisub issue 177

    7. Quick Fix Checklist¶

    Crunchyroll Subtitle Restoration Checklist

    • LayoutRes = PlayRes
    • Old CR style restored (Trebuchet MS, 60pt, 7px outline, etc.)
    • All required fonts attached
    • Signs properly typeset in Aegisub
    • Subtitle playback tested in mpv, VLC, Plex
